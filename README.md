# 📷 Nexor (Landing Page)

Uma landing page moderna e responsiva desenvolvida para divulgação de uma agência.  
O projeto foi criado com foco em experiência do usuário, design responsivo e interatividade durante a navegação.

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

---

## 🎯 Funcionalidades

- Layout totalmente responsivo  
- Interface moderna e intuitiva  
- Navegação suave entre seções  
- Animações com scroll  
- Aparição dinâmica de elementos durante a navegação  
- Troca automática de textos utilizando JavaScript (input)  
- Reset dos campos de input ao clicar 

---

## 📱 Responsividade

O projeto foi desenvolvido utilizando técnicas de responsividade para garantir uma boa experiência em:

- Desktop
- Tablets
- Smartphones

---

## ⚙️ Funcionalidades em JavaScript

O JavaScript foi utilizado para:

- Criar animações de entrada ao rolar a página
- Exibir elementos dinamicamente
- Alterar textos automaticamente
- Resetar os campos dos formulários durante a interação do usuário
